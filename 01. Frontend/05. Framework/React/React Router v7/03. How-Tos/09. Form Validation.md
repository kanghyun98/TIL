# Form Validation

React Router에서는 폼 검증을 **action을 통해 처리하는 방식**을 사용합니다.  
폼이 제출되면 해당 라우트의 `action`이 실행되고, 서버에서 데이터를 검증한 뒤 결과를 UI에 전달합니다.  

<br>

## 1. Setting Up

먼저 기본적인 signup 라우트 및 signup 컴포넌트를 생성합니다.

```ts
import {
  type RouteConfig,
  route,
} from "@react-router/dev/routes";

export default [
  route("signup", "signup.tsx"),
] satisfies RouteConfig;
```

```tsx
import type { Route } from "./+types/signup";
import { useFetcher } from "react-router";

export default function Signup(_: Route.ComponentProps) {
  let fetcher = useFetcher();

  return (
    <fetcher.Form method="post">
      <p>
        <input type="email" name="email" />
      </p>

      <p>
        <input type="password" name="password" />
      </p>

      <button type="submit">Sign Up</button>
    </fetcher.Form>
  );
}
```

이 폼이 제출되면 해당 라우트의 `action`이 실행됩니다. 

<br>

## 2. Add the Action

폼 데이터를 검증하기 위해 `action`을 추가합니다.

```ts
// signup.tsx
import type { Route } from "./+types/signup";
import { redirect, useFetcher, data } from "react-router";

export default function Signup(_: Route.ComponentProps) {
  // omitted for brevity
}

export async function action({
  request,
}: Route.ActionArgs) {
  const formData = await request.formData();
  const email = String(formData.get("email"));
  const password = String(formData.get("password"));

  const errors = {};

  if (!email.includes("@")) {
    errors.email = "Invalid email address";
  }

  if (password.length < 12) {
    errors.password =
      "Password should be at least 12 characters";
  }

  if (Object.keys(errors).length > 0) {
    return data({ errors }, { status: 400 });
  }

  // Redirect to dashboard if validation is successful
  return redirect("/dashboard");
}
```

`action`에서는 다음 작업이 수행됩니다.

1. `request.formData()`로 폼 데이터 읽기  
2. 서버에서 입력값 검증 
3. 검증 실패 시 오류 객체 반환  
4. 검증 성공 시 정상 결과 반환  

<br>

## 3. Display Validation Errors

`useActionData()`를 사용하면 action이 반환한 데이터를 컴포넌트에서 사용할 수 있습니다.

```tsx
export default function Signup(_: Route.ComponentProps) {
  let fetcher = useFetcher();
  let errors = fetcher.data?.errors;
  return (
    <fetcher.Form method="post">
      <p>
        <input type="email" name="email" />
        {errors?.email ? <em>{errors.email}</em> : null}
      </p>

      <p>
        <input type="password" name="password" />
        {errors?.password ? (
          <em>{errors.password}</em>
        ) : null}
      </p>

      <button type="submit">Sign Up</button>
    </fetcher.Form>
  );
}
```

이렇게 하면 서버에서 반환된 오류 메시지를 UI에 표시할 수 있습니다.
