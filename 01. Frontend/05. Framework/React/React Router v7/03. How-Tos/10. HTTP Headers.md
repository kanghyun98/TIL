# HTTP Headers

React Router에서는 라우트 단위로 **HTTP 헤더를 설정할 수 있습니다.**  

헤더는 주로 **route module의 `headers` export**를 통해 정의 가능하며, `entry.server.tsx`에서도 설정할 수 있습니다.

<br>

## 1. From Route Modules

가장 기본적인 방법은 라우트 모듈에서 `headers` 함수를 export하는 것입니다.

> **파일 단위로 헤더를 설정**

```ts
// react-router
// some-route.tsx
export function headers() {
  return {
    "Content-Security-Policy": "default-src 'self'",
    "X-Frame-Options": "DENY",
    "X-Content-Type-Options": "nosniff",
    "Cache-Control": "max-age=3600, s-maxage=86400",
  };
}

// express
res.setHeader("Cache-Control", "max-age=3600");
```

`headers` 함수는 다음 두 가지 형태를 반환할 수 있습니다.

- `Headers` 
- `HeadersInit` 

<br>

## 2. From loaders and actions

헤더가 **데이터에 따라 달라져야 하는 경우**, `loader`나 `action`에서 설정할 수 있습니다.

이때는 `data()`를 사용해 반환값에 헤더를 포함시킵니다.

```ts
import { data } from "react-router";

export async function loader({ params }) {
  let [page, ms] = await fakeTimeCall(
    await getPage(params.id),
  );

  return data(page, {
    headers: {
      "Server-Timing": `page;dur=${ms}`,
    },
  });
}
```

<br>

### 2.1 headers export로 반환해야 함

`loader`나 `action`에서 설정한 헤더는 자동으로 응답에 포함되지 않습니다.  
반드시 `headers` 함수에서 반환해야 합니다.

```ts
function hasAnyHeaders(headers: Headers): boolean {
  return [...headers].length > 0;
}

export function headers({
  actionHeaders,
  loaderHeaders,
}: HeadersArgs) {
  return hasAnyHeaders(actionHeaders)
    ? actionHeaders
    : loaderHeaders;
}
```

<br>

### 2.2 예외: Set-Cookie

Set-Cookie 헤더는 예외적으로 부모 라우트에서 설정한 값이 자식 라우트에서도 자동으로 유지됩니다.

>  child가 headers를 정의하면 parent headers는 사라집니다.

<br>

## 3. Merging with parent headers

중첩 라우트에서는 부모와 자식이 각각 헤더를 설정할 수 있습니다. 기본적으로는 **가장 깊은 라우트의 헤더만 적용됩니다.**

```ts
route("pages", "pages-layout-with-nav.tsx", [
  route(":slug", "page.tsx"),
]);
```

<br>

### 3.1 Appending

부모 헤더를 유지하면서 값을 추가하려면 `append`를 사용합니다.

```ts
export function headers({ parentHeaders }) {
  parentHeaders.append(
    "Permissions-Policy",
    "geolocation=()"
  );
  return parentHeaders;
}
```

<br>

### 3.2 Setting

부모 헤더를 덮어쓰려면 `set`을 사용합니다.

```ts
export function headers({ parentHeaders }) {
  parentHeaders.set(
    "Cache-Control",
    "max-age=3600"
  );
  return parentHeaders;
}
```

<br>

## 4. From `entry.server.tsx`

애플리케이션 전체에 적용되는 **글로벌 헤더**는 `entry.server.tsx`에서 설정할 수 있습니다.

```ts
export default async function handleRequest(
  request,
  responseStatusCode,
  responseHeaders,
  routerContext,
  loadContext,
) {
  // set, append global headers
  responseHeaders.set(
    "X-App-Version",
    routerContext.manifest.version,
  );

  return new Response(await getStream(), {
    headers: responseHeaders,
    status: responseStatusCode,
  });
}
```
