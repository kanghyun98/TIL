# Status Codes

loader와 action에서 `data`로 상태 코드를 설정할 수 있습니다.

```tsx
// route('/projects/:projectId', './project.tsx')
import type { Route } from "./+types/project";
import { data } from "react-router";
import { fakeDb } from "../db";

export async function action({
  request,
}: Route.ActionArgs) {
  let formData = await request.formData();
  let title = formData.get("title");

  if (!title) {
    return data(
      { message: "Invalid title" },
      { status: 400 },
    );
  }

  if (!projectExists(title)) {
    let project = await fakeDb.createProject({ title });
    return data(project, { status: 201 });
  } else {
    let project = await fakeDb.updateProject({ title });
    // the default status code is 200, no need for `data`
    return project;
  }
}
```

이와 같은 폼 에러 렌더링에 대한 자세한 내용은 Form Validation을 참고하세요.

<br>

자주 사용하는 또 다른 상태 코드는 404입니다.

```tsx
// route('/projects/:projectId', './project.tsx')
import type { Route } from "./+types/project";
import { data } from "react-router";
import { fakeDb } from "../db";

export async function loader({ params }: Route.ActionArgs) {
  let project = await fakeDb.getProject(params.id);

  if (!project) {
    // throw to ErrorBoundary
    throw data(null, { status: 404 });
  }

  return project;
}
```

throw된 `data`에 대한 자세한 내용은 Error Boundaries를 참고하세요.
