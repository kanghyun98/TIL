# Security

이 문서는 결코 포괄적인 가이드는 아니지만, Security라는 매우 큰 범주 아래에서 몇 가지 측면을 다루는 데 도움이 되는 기능을 제공합니다.

<br>

## 1. `Content-Security-Policy`

애플리케이션에 `Content-Security-Policy (CSP)`를 구현하고 있고, 특히 `unsafe-inline` 지시어를 사용하는 경우에는 HTML에 렌더링되는 인라인 `<script>` 요소에 `nonce` 속성을 지정해야 합니다.

이는 인라인 스크립트를 생성하는 모든 API에 지정되어야 하며, 다음이 포함됩니다.

- `<Scripts nonce>` (`root.tsx`)
- `<ScrollRestoration nonce>` (`root.tsx`)
- `<ServerRouter nonce>` (`entry.server.tsx`)
- `renderToPipeableStream(..., { nonce })` (`entry.server.tsx`)
- `renderToReadableStream(..., { nonce })` (`entry.server.tsx`)
