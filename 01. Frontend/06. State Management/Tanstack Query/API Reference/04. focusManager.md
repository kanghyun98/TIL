# focusManager

focusManager는 TanStack Query 내부의 focus 상태를 관리합니다.

focusManager는 다음 용도로 사용할 수 있습니다.

- 기본 이벤트 리스너 변경
- focus 상태를 수동으로 변경



사용 가능한 메서드는 다음과 같습니다.

- `setEventListener`
- `subscribe`
- `setFocused`
- `isFocused`

<br>

## 1. focusManager.setEventListener

`setEventListener`는 커스텀 이벤트 리스너를 설정할 때 사용합니다.

```tsx
import { focusManager } from '@tanstack/react-query'

focusManager.setEventListener((handleFocus) => {
  // Listen to visibilitychange
  if (typeof window !== 'undefined' && window.addEventListener) {
    window.addEventListener('visibilitychange', handleFocus, false)
  }

  return () => {
    // Be sure to unsubscribe if a new handler is set
    window.removeEventListener('visibilitychange', handleFocus)
  }
})
```

<br>

## 2. focusManager.subscribe

`subscribe`는 visibility 상태 변경을 구독할 때 사용합니다.

이 메서드는 unsubscribe 함수를 반환합니다.

```tsx
import { focusManager } from '@tanstack/react-query'

const unsubscribe = focusManager.subscribe((isVisible) => {
  console.log('isVisible', isVisible)
})
```

<br>

## 3. focusManager.setFocused

`setFocused`는 focus 상태를 수동으로 설정할 때 사용합니다.

`undefined`를 설정하면 기본 focus check로 돌아갑니다.

```tsx
import { focusManager } from '@tanstack/react-query'

// Set focused
focusManager.setFocused(true)

// Set unfocused
focusManager.setFocused(false)

// Fallback to the default focus check
focusManager.setFocused(undefined)
```

Options

- `focused: boolean | undefined`

<br>

## 4. focusManager.isFocused

`isFocused`는 현재 focus 상태를 가져올 때 사용합니다.

```tsx
const isFocused = focusManager.isFocused()
```
