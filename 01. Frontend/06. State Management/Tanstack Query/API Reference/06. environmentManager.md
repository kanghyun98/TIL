# environmentManager

`environmentManager`는 TanStack Query가 현재 런타임을 서버 환경으로 처리할지 감지하는 방식을 관리한다.

기본적으로 `query-core`에서 export된 `isServer` 유틸리티와 동일한 서버 감지 방식을 사용한다.

전통적인 브라우저/서버 환경이 아닌 런타임에서는 이 매니저를 사용해 서버 감지를 전역으로 재정의할 수 있다.



사용 가능한 메서드는 다음과 같다.

- `isServer`
- `setIsServer`

<br>

## 1. environmentManager.isServer

현재 런타임이 서버 환경으로 처리되는지 반환한다.

```tsx
import { environmentManager } from '@tanstack/react-query'

const server = environmentManager.isServer()
```

<br>

## 2. environmentManager.setIsServer

서버 체크를 전역으로 재정의한다.

```tsx
import { environmentManager } from '@tanstack/react-query'

// Override
environmentManager.setIsServer(() => {
  return typeof window === 'undefined' && !('chrome' in globalThis)
})
```

옵션:

- `isServerValue: () => boolean`

기본 동작으로 되돌리려면 함수를 `query-core`의 `isServer` 유틸리티로 다시 설정한다.

```tsx
import { environmentManager, isServer } from '@tanstack/react-query'

environmentManager.setIsServer(() => isServer)
```
