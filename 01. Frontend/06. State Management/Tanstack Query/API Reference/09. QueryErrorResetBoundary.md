# QueryErrorResetBoundary

`QueryErrorResetBoundary`는 `suspense` 또는 `throwOnError`를 사용하는 쿼리에서 에러 이후 다시 시도할 수 있도록 도와주는 컴포넌트입니다.

에러가 발생한 뒤 리렌더링될 때 쿼리에 다시 시도하고 싶다는 것을 알려줄 방법이 필요합니다.

`QueryErrorResetBoundary` 컴포넌트를 사용하면 컴포넌트 경계 안에 있는 모든 쿼리 에러를 reset할 수 있습니다.

```tsx
import { QueryErrorResetBoundary } from '@tanstack/react-query'
import { ErrorBoundary } from 'react-error-boundary'

const App = () => (
  <QueryErrorResetBoundary>
    {({ reset }) => (
      <ErrorBoundary
        onReset={reset}
        fallbackRender={({ resetErrorBoundary }) => (
          <div>
            There was an error!
            <Button onClick={() => resetErrorBoundary()}>Try again</Button>
          </div>
        )}
      >
        <Page />
      </ErrorBoundary>
    )}
  </QueryErrorResetBoundary>
)
```
