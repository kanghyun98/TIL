# Filters

TanStack Query의 일부 메서드는 `QueryFilters` 또는 `MutationFilters` 객체를 받습니다.

<br>

## 1. Query Filters

Query Filter는 특정 조건으로 Query를 매칭하기 위한 객체입니다.

```tsx
// Cancel all queries
await queryClient.cancelQueries()

// Remove all inactive queries that begin with `posts` in the key
queryClient.removeQueries({ queryKey: ['posts'], type: 'inactive' })

// Refetch all active queries
await queryClient.refetchQueries({ type: 'active' })

// Refetch all active queries that begin with `posts` in the key
await queryClient.refetchQueries({ queryKey: ['posts'], type: 'active' })
```

Query Filter 객체는 다음 속성을 지원합니다.

- `queryKey?: QueryKey`
  - 매칭할 Query Key를 정의합니다.

- `exact?: boolean`
  - Query Key를 포괄적으로 검색하지 않으려면 `exact: true`를 전달합니다.
  - 전달한 Query Key와 정확히 일치하는 Query만 반환합니다.

- `type?: 'active' | 'inactive' | 'all'`
  - 기본값은 `all`입니다.
  - `active`로 설정하면 active query를 매칭합니다.
  - `inactive`로 설정하면 inactive query를 매칭합니다.

- `stale?: boolean`
  - `true`로 설정하면 stale query를 매칭합니다.
  - `false`로 설정하면 fresh query를 매칭합니다.

- `fetchStatus?: FetchStatus`
  - `fetching`으로 설정하면 현재 fetching 중인 Query를 매칭합니다.
  - `paused`로 설정하면 fetch를 원했지만 pause된 Query를 매칭합니다.
  - `idle`로 설정하면 fetching 중이 아닌 Query를 매칭합니다.

- `predicate?: (query: Query) => boolean`
  - 모든 매칭된 Query에 마지막 필터로 사용되는 predicate 함수입니다.
  - 다른 필터가 지정되지 않으면 캐시의 모든 Query에 대해 평가됩니다.

<br>

## 2. Mutation Filters

Mutation Filter는 특정 조건으로 Mutation을 매칭하기 위한 객체입니다.

```tsx
// Get the number of all fetching mutations
await queryClient.isMutating()

// Filter mutations by mutationKey
await queryClient.isMutating({ mutationKey: ['post'] })

// Filter mutations using a predicate function
await queryClient.isMutating({
  predicate: (mutation) => mutation.state.variables?.id === 1,
})
```

Mutation Filter 객체는 다음 속성을 지원합니다.

- `mutationKey?: MutationKey`
  - 매칭할 Mutation Key를 정의합니다.

- `exact?: boolean`
  - Mutation Key를 포괄적으로 검색하지 않으려면 `exact: true`를 전달합니다.
  - 전달한 Mutation Key와 정확히 일치하는 Mutation만 반환합니다.

- `status?: MutationStatus`
  - Mutation의 status에 따라 필터링할 수 있습니다.

- `predicate?: (mutation: Mutation) => boolean`
  - 모든 매칭된 Mutation에 마지막 필터로 사용되는 predicate 함수입니다.
  - 다른 필터가 지정되지 않으면 캐시의 모든 Mutation에 대해 평가됩니다.

<br>

## 3. Utils

### 3.1 matchQuery

```tsx
const isMatching = matchQuery(filters, query)
```

제공된 Query Filter 집합과 Query가 매칭되는지를 boolean으로 반환합니다.

<br>

### 3.2 matchMutation

```tsx
const isMatching = matchMutation(filters, mutation)
```

제공된 Mutation Filter 집합과 Mutation이 매칭되는지를 boolean으로 반환합니다.
