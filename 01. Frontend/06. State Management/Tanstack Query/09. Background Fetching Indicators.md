# Background Fetching Indicators

쿼리의 `status === 'pending'` 상태만으로도 초기 하드 로딩 상태를 표시할 수 있습니다.

다만, 쿼리가 백그라운드에서 다시 가져오는 중이라는 추가 표시가 필요할 때도 있습니다.

이를 위해 쿼리는 `isFetching` 불리언도 제공합니다.

- `status` 상태와 관계없이 사용할 수 있습니다.
- 현재 fetching 중인지 표시할 때 사용할 수 있습니다.

```tsx
function Todos() {
  const {
    status,
    data: todos,
    error,
    isFetching,
  } = useQuery({
    queryKey: ['todos'],
    queryFn: fetchTodos,
  })

  return status === 'pending' ? (
    <span>Loading...</span>
  ) : status === 'error' ? (
    <span>Error: {error.message}</span>
  ) : (
    <>
      {isFetching ? <div>Refreshing...</div> : null}
      <div>
        {todos.map((todo) => (
          <Todo todo={todo} />
        ))}
      </div>
    </>
  )
}
```

## 1. Displaying Global Background Fetching Loading State

개별 쿼리의 로딩 상태와 별개로, 어떤 쿼리든 fetching 중일 때 전역 로딩 인디케이터를 표시하고 싶다면 `useIsFetching` 훅을 사용할 수 있습니다.

```tsx
import { useIsFetching } from '@tanstack/react-query'

function GlobalLoadingIndicator() {
  const isFetching = useIsFetching()

  return isFetching ? (
    <div>Queries are fetching in the background...</div>
  ) : null
}
```
